# Creator Skill Reference

Full reference material — loaded on demand, not on every activation.

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Success |
| 1 | General error |
| 2 | Invalid arguments |
| 3 | Not authenticated |
| 4 | Resource not found |
| 5 | Conflict / not ready |
| 6 | Validation error |

## Environment Variables

| Variable | Default | Description |
|---------|---------|-------------|
| `AGNT_API_BASE` | `https://api.agnt-gm.ai/api` | API base URL |
| `AGNT_CREDENTIALS_DIR` | `~/.agnt` | Credentials directory |
